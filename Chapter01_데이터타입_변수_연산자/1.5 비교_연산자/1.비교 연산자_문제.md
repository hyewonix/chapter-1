# 비교 연산자
비교 연산자를 활용해 값을 비교하세요.

## 설명
- 두 개의 정수를 입력받아 **각 비교 연산자(==, !=, >, <, >=, <=)**의 결과를 출력합니다.
- `if` 문을 사용하지 않고, 비교 연산의 결과를 직접 출력합니다.
- 비교 연산 결과는 `True` 또는 `False`로 출력됩니다.

## 입력 예제
10  
3

## 출력
10 == 3: False  
10 != 3: True  
10 > 3: True  
10 < 3: False  
10 >= 3: True  
10 <= 3: False
