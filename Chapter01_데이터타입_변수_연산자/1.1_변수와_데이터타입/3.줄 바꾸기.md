# 줄 바꾸기

## `print()` 함수 사용
- `print()`함수를 사용해 아래 예제와 같이 줄을 바꿀 수 있습니다. 
- 코드는 기본적으로 위에서 아래로 실행됩니다. 

### 입력
```python
print("Hello, World!")
print("Python is fun.")
```
### 출력
Hello, World!
Python is fun.


## ''' 또는 """ 사용하기
- 작은 따옴표와 큰 따옴표를 **세 번 연속**으로 사용하여 줄을 바꿀 수 있습니다. 

### 입력
```python
print("""Hello, World!
Python is fun.""")
```
### 출력
Hello, World!
Python is fun.


## \n 이스케이프 문자 사용하기
- 줄을 바꾸고 싶은 곳에 `\n`을 사용합니다.

### 입력
```python
print('Hello, World!\nPython is fun.')
```
### 출력
Hello, World!
Python is fun.
