# Purpose
- `type()` 함수를 사용하여 변수에 저장된 데이터의 타입을 확인하는 방법을 학습합니다.

# Solution
- `type(변수)`를 사용하면 해당 변수의 데이터 타입을 확인할 수 있습니다.  
- `print(type(변수))`를 사용하여 타입 정보를 출력합니다.  

# Input
```python
# 정수 데이터 타입 확인
a = 25
print(type(a))  # <class 'int'>
# 실수 데이터 타입 확인
b = 7.89
print(type(b))  # <class 'float'>
# 문자열 데이터 타입 확인
c = "Python"
print(type(c))  # <class 'str'>
```

# Output
<class 'int'>
<class 'float'>
<class 'str'>
